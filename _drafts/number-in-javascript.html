Number in javascript.

Hi, today let's dive into how number work in javascript.

As we can see In javascript every number are float64 following the IEEE 754-2008 format, in this post we will see how IEEE 754-2008 work.

The designer of the IEEE-754 format choose to represent number in a scientifique notation, So 324.569 become 3.24569 * 10^3, since we are on a computer we must deal with binary thus this number must be represented in base 2

Number are represented in binary as such:

they are composed of 3 part:

1 bit for the sign wich give the sign of the number, 0 for positive, 1 for negative

11 bit for the exponent[1]

52 bit for the mantissa[2]

So how our number 324.569 fit in those bit?
